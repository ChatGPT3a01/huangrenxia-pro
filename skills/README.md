# SKILL 技能管理

這個資料夾是「隨身黃仁蝦AI系統」的技能包管理目錄。

## 目錄結構

- `import/`：將技能包（.zip 或資料夾）放入這裡
- `installed/`：系統已安裝的技能

## 如何安裝技能

1. 將技能包 .zip 或技能資料夾放入 `import/`
2. 從主選單選擇 **[8] 技能管理**
3. 選擇 **[2] 安裝新技能**
4. 選擇要安裝的技能包

## 如何移除技能

1. 從主選單選擇 **[8] 技能管理**
2. 選擇 **[3] 移除技能**
3. 選擇要移除的技能

## SKILL.md 標準格式

每個技能包**必須**包含一個 `SKILL.md` 檔案，建議格式如下：

```markdown
---
name: my-skill
description: 這個技能的一句話說明
version: 1.0
author: 作者名
---

# 技能名稱

## 功能說明
描述這個技能能做什麼...

## 使用範例
如何使用這個技能...
```

### Front Matter 欄位

| 欄位 | 必要？ | 說明 |
|------|--------|------|
| name | 建議 | 技能名稱（英文，用作識別） |
| description | 建議 | 一句話描述 |
| version | 選用 | 版本號（如 1.0、2.1） |
| author | 選用 | 作者名稱 |

## 技能包結構範例

```
my-skill/
  SKILL.md          ← 必要（技能描述與使用說明）
  scripts/          ← 選用（腳本檔案）
  data/             ← 選用（技能資料）
  README.md         ← 選用（額外說明）
```
