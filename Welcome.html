三步開始使用

啟動服務

Windows：插入 USB，雙擊 點這個啟動-隨身黃仁蝦AI-Windows.bat
Mac：插入 USB，雙擊 Mac-啟動.command

設定 AI 模型

開啟 Config.html 或從主選單選「設定 API Key」。
選擇你想用的 AI 供應商，填入 API Key 即可。

開始對話

啟動後瀏覽器會自動開啟 http://127.0.0.1:18789。
你的 AI 助手已就緒，開始提問吧！

重要觀念：USB = 身份鑰匙（認證用），電腦 = AI 主機（運算用）。啟動時必須插著 USB。

常見問題

啟動失敗怎麼辦？

確認 USB 有插著，然後從主選單選「查看 OpenClaw 狀態」檢查。

如何更換 AI 模型？

從主選單選「設定（API Key / 通訊頻道 / 密碼）」，或開啟 Config.html。

可以接 LINE / Telegram 嗎？

可以！在首次設定或主選單的「設定」中填入通訊頻道資訊即可。
