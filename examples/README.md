## 如何使用

1. 從主選單選擇 **[6] 使用教學 / 案例庫**
2. 選擇 **[2] 瀏覽使用案例**
3. 選擇分類 → 選擇範例 → 查看情境和 Prompt

## 範例格式

每個 `.md` 檔案包含：
- **情境**：什麼時候會用到
- **示範 Prompt**：可以直接複製貼上的 AI 指令
- **預期效果**：AI 會回覆什麼

## 新增自訂範例

您可以自行新增 `.md` 檔案到對應分類資料夾中，格式參考現有範例即可。
