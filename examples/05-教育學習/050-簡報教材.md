# 簡報教材製作

## 情境
你需要製作一份教育訓練用的簡報教材，要在公司內部分享某個專業主題。你希望簡報不只是文字堆疊，而是有清楚的架構、吸引人的視覺建議，以及能促進聽眾互動的教學設計。

## 示範 Prompt
> 我要做一份「新人到職資安教育訓練」簡報，對象是公司新進員工（非技術背景），時間 40 分鐘。請幫我：1) 規劃簡報架構，大約 25-30 頁，含每頁的標題和要點（每頁不超過 3 個重點）；2) 每 5-8 頁插入一個互動環節（小測驗、案例討論或情境模擬）；3) 建議每頁適合搭配的視覺元素（圖示類型、配色建議）；4) 提供 3 個真實發生過的台灣企業資安事件作為案例教材；5) 最後一頁設計成重點回顧的懶人包格式。

## 預期效果
AI 會產出：
- 完整的 25-30 頁簡報架構，每頁含標題、要點與講者備註
- 穿插在適當位置的互動環節設計（含題目與引導語）
- 視覺設計建議（圖示風格、配色方案、版面配置）
- 台灣本土資安案例整理，含事件摘要與教訓分析
