# 數據視覺化圖表

## 情境
你準備在季度經營會議上簡報公司的營運數據。主管曾批評過去的報告「數字太多、圖表太醜、看不出重點」。你希望 AI 能建議最適合的圖表類型、配色方案，並提供可以直接使用的圖表程式碼或設計指引。

## 示範 Prompt
> 我要在經營會議上用簡報呈現以下數據，請幫我選擇最佳的圖表類型並提供 Python matplotlib 程式碼：1) 過去 12 個月的營收趨勢（含去年同期對比）；2) 五個產品線的營收佔比；3) 北中南東四區的業績達成率（目標 vs. 實際）；4) 客戶滿意度的五個面向分數（雷達圖）；5) 各月份新客戶數與流失客戶數的對比。每張圖請：使用商務風格配色（藍灰色系為主）、字型大小適合投影、標註關鍵數據點、加上圖表標題與圖例說明。請提供完整可執行的 Python 程式碼，並說明每張圖為什麼選這種圖表類型。

## 預期效果
AI 會產出：
- 5 種圖表的類型選擇理由（如：趨勢用折線圖、佔比用圓餅圖或甜甜圈圖）
- 完整可執行的 Python matplotlib/seaborn 程式碼，含範例數據
- 商務風格的配色方案（含色碼值）和字型設定
- 圖表設計原則說明（如：資料墨水比、去除不必要的裝飾線條）
