# SQL 查詢

## 情境
需要從資料庫中撈取特定條件的資料，但 SQL 語法不熟或查詢邏輯較複雜。用白話文描述你想要的結果，讓 AI 幫你撰寫正確的 SQL 查詢語句。

## 示範 Prompt
> 我有一個電商資料庫，包含以下資料表：
> - orders（訂單）：order_id, customer_id, order_date, total_amount, status
> - customers（客戶）：customer_id, name, email, city, created_at
> - order_items（訂單明細）：item_id, order_id, product_id, quantity, unit_price
>
> 請幫我寫一個 SQL 查詢：找出 2025 年每個月的銷售額前 3 名的城市，顯示城市名稱、月份、訂單數量、總銷售額，並按月份和銷售額排序。

## 預期效果
AI 會產出：
- 使用 Window Function（ROW_NUMBER）的完整 SQL 查詢語句
- 查詢邏輯的逐步拆解說明，解釋 CTE、GROUP BY、排名函數的用法
- 效能優化建議，例如建議在 order_date 和 city 欄位加上索引
