# Python 腳本

## 情境
日常工作中有許多重複性的任務，例如批次處理檔案、整理資料、產生報表等。讓 AI 幫你快速撰寫 Python 腳本，把手動作業自動化。

## 示範 Prompt
> 請幫我寫一個 Python 腳本，功能如下：
> 1. 掃描指定資料夾中所有的 Excel 檔案（.xlsx）
> 2. 讀取每個檔案的第一個工作表
> 3. 合併所有檔案的資料到一個 DataFrame
> 4. 新增一欄「來源檔案」記錄每筆資料來自哪個檔案
> 5. 移除完全重複的列
> 6. 輸出成一個合併後的 Excel 檔案，並自動調整欄寬
>
> 請使用 openpyxl 和 pandas，並加入錯誤處理和進度顯示。

## 預期效果
AI 會產出：
- 可直接執行的完整 Python 腳本，包含必要的 import 和主程式區塊
- 錯誤處理機制：檔案讀取失敗時跳過並記錄，不中斷整個流程
- 使用說明與執行方式，包含所需套件的安裝指令（pip install）
