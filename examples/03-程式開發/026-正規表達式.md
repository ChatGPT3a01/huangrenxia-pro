# 正規表達式

## 情境
需要從大量文字中擷取特定格式的資料（如電話號碼、Email、身分證字號等），或是進行複雜的字串比對與取代。正規表達式語法難記，讓 AI 幫你寫出正確的 pattern。

## 示範 Prompt
> 請幫我寫出以下情境的正規表達式（使用 Python re 模組），並附上測試範例：
>
> 1. 驗證台灣手機號碼（09 開頭，共 10 碼，中間可能有 - 分隔）
> 2. 擷取文章中所有的 Email 地址
> 3. 從 HTML 原始碼中擷取所有 img 標籤的 src 屬性值
> 4. 將日期格式從「2025/03/15」轉換為「2025-03-15」
>
> 每個都請附上 pattern 說明和 Python 程式碼範例。

## 預期效果
AI 會產出：
- 每個需求的正規表達式 pattern，附上每個符號的中文解釋
- 可執行的 Python 程式碼範例，包含 re.match、re.findall、re.sub 等用法
- 邊界情況提醒，例如手機號碼的 +886 國際格式、Email 中的特殊字元等
