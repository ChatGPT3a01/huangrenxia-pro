# 程式碼解釋

## 情境
接手同事的專案或閱讀開源程式碼時，遇到一段看不懂的邏輯。把程式碼貼給 AI，讓它用白話文逐行解釋，幫助你快速理解程式運作方式。

## 示範 Prompt
> 請用繁體中文逐行解釋以下 JavaScript 程式碼的功能，並說明整體用途：
>
> ```javascript
> const debounce = (fn, delay) => {
>   let timer = null;
>   return (...args) => {
>     clearTimeout(timer);
>     timer = setTimeout(() => fn.apply(this, args), delay);
>   };
> };
>
> const handleSearch = debounce((keyword) => {
>   fetch(`/api/search?q=${encodeURIComponent(keyword)}`)
>     .then(res => res.json())
>     .then(data => renderResults(data));
> }, 300);
> ```

## 預期效果
AI 會產出：
- 逐行中文註解，解釋每一行程式碼的作用
- 整體功能說明：這是一個防抖（debounce）搜尋功能，避免使用者快速輸入時頻繁發送 API 請求
- 實務應用場景舉例，例如搜尋框即時搜尋、視窗 resize 事件處理等
