# 除錯幫手

## 情境
開發過程中遇到程式錯誤，看了半天找不到問題所在。把錯誤訊息和相關程式碼貼給 AI，讓它幫你快速定位問題並提供修復建議。

## 示範 Prompt
> 我的 Python Flask 程式出現以下錯誤，請幫我找出原因並提供修正方式：
>
> ```
> TypeError: 'NoneType' object is not subscriptable
> File "app.py", line 45, in get_user
>     return data['user']['name']
> ```
>
> 相關程式碼：
> ```python
> @app.route('/user/<int:user_id>')
> def get_user(user_id):
>     data = db.execute("SELECT * FROM users WHERE id = ?", (user_id,)).fetchone()
>     return data['user']['name']
> ```

## 預期效果
AI 會產出：
- 錯誤原因分析：查詢結果為 None 時直接取值導致錯誤
- 具體修正程式碼：加入 None 檢查與錯誤處理
- 防禦性寫法建議：使用 `.get()` 方法或 try-except 避免未來類似問題
