# 網頁前端

## 情境
需要快速建立網頁介面元件，例如表單、資料表格、導覽列等。讓 AI 根據你的需求產生 HTML、CSS、JavaScript 程式碼，加速前端開發。

## 示範 Prompt
> 請幫我用純 HTML + CSS + JavaScript（不使用框架）建立一個後台管理的資料表格元件，需要以下功能：
>
> 1. 響應式設計，手機版自動改為卡片式排版
> 2. 欄位排序功能（點擊表頭可升降排序）
> 3. 關鍵字搜尋過濾
> 4. 分頁顯示，每頁 10 筆
> 5. 整體風格簡潔現代，使用深藍配色
>
> 資料欄位：訂單編號、客戶名稱、訂單日期、金額、狀態（待處理/已出貨/已完成）

## 預期效果
AI 會產出：
- 完整的 HTML 結構、CSS 樣式、JavaScript 邏輯，可直接存檔開啟使用
- 響應式斷點設計，桌面版為表格、手機版為卡片式排版
- 互動功能完整：排序箭頭指示、搜尋即時過濾、分頁按鈕含頁碼顯示
