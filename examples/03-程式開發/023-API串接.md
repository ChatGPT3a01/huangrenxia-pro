# API 串接

## 情境
需要串接第三方服務的 API（如金流、物流、社群登入），但官方文件不夠清楚或範例太少。讓 AI 根據 API 規格幫你產生串接程式碼，加速開發流程。

## 示範 Prompt
> 我要用 Python requests 串接綠界科技（ECPay）的信用卡付款 API。請幫我寫一個完整的串接範例，包含：
> 1. 產生交易所需的檢查碼（CheckMacValue）
> 2. 建立訂單並送出付款請求
> 3. 接收付款結果的回呼（callback）處理
>
> 測試環境的 MerchantID 是 3002607，HashKey 是 pwFHCqoQZGmho4w6，HashIV 是 EkRm7iFT261dpevs。

## 預期效果
AI 會產出：
- 完整的 Python 串接程式碼，包含 CheckMacValue 的 SHA256 加密計算
- Flask 路由範例：建立訂單的前端頁面與接收回呼的 endpoint
- 常見錯誤排除提示，例如編碼問題、參數排序錯誤等注意事項
