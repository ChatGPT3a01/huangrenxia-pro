# 操作手冊 PDF 產生方式

本資料夾包含兩份 HTML 文件，可透過 Chrome 瀏覽器轉為 PDF。

## 檔案清單

| 檔案 | 用途 | 對外？ |
|------|------|--------|
| `index.html` | 客戶操作手冊（28 頁） | 對外（交付給客戶） |
| `comparison.html` | 競品比較分析（含浮水印） | 內部機密，禁止外流 |
| `style.css` | 共用樣式 | - |

## 匯出 PDF 步驟

1. 用 **Chrome** 瀏覽器開啟 HTML 檔案
2. 按 `Ctrl + P`（列印）
3. 目的地選「**另存為 PDF**」
4. 紙張大小選 **A4**
5. 邊界選「**預設**」
6. **勾選「背景圖形」**（重要！否則封面和配色不會顯示）
7. 按「儲存」

## 替換截圖

手冊中的截圖目前是灰色虛線佔位框。要替換為實際截圖：

1. 將截圖圖片放入 `screenshots/` 資料夾
2. 在 `index.html` 中找到對應的 `<div class="screenshot">` 區塊
3. 將內容替換為 `<img src="screenshots/檔名.png" alt="說明">`
4. 重新列印 PDF
